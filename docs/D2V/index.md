# Dealer to Vendor
**D2V** (Dealer to Vendor) is the Dysel integration layer that connects Business Central directly to your parts vendors. It handles authentication, request formatting, and response processing so that your team can check prices, confirm availability, and submit orders without leaving Business Central. Check the documentation for the specific vendors we support for more details on how to set up and use their integrations.
