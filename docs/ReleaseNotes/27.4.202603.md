| 67112 | Equipment | Added a new option in field Model Type of the equipment model. New option is called 'Placeholder' and meant to be a sort of dummy/temporary model to choose when creating an equipment object. To start using a Placeholder model, a model specifically created as Placeholder will need to be created. This dummy/temporary model should be replaced by the actual model in a later stage. If entries exist for an object with a good model, Placeholder cannot be added. The Placeholder model is not intended to be the model of an object for its entire life cycle. 

There are several restrictions on placeholder models, you cannot create most of the entries for example. |
