# Maintenance Contracts
Maintenance contracts help users set up and manage ongoing service agreements with customers. They allow you to track what services are covered, how long the agreement lasts, and when customers should be billed. By using maintenance contracts, users can automate recurring invoices, monitor contract start and end dates, and ensure customers receive the services they are entitled to without manual follow‑up.
