# Setup
This section covers the configuration pages required to prepare Dysel BC for use within your organization. The setup pages described here are typically completed during initial implementation and should be reviewed by a system administrator or Dysel consultant before the system is put into production.

Each page controls a specific area of Dysel BC's Rental behavior. Settings configured here apply system-wide and affect how documents, processes, and integrations operate across all users and the Service module.
